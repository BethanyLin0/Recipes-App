# Recipes-App
This is an iOS application that runs and stores locally on the user’s MacBook, allowing the user to create and record ice cream recipes and helps with portions calculations and budgeting. The goal is to make the process of ice cream-making more convenient and efficient for the user.

Steps:
1. Download the “Recipes-App” Folder
2. Create a new folder in Xcode with the following folders/files
<img width="392" height="598" alt="Screenshot 2025-12-11 at 19 15 13" src="https://github.com/user-attachments/assets/91e79df9-0270-45b4-8a1b-db8afc6b4b1a" />

3. Copy/paste the codes from each file in the folder to the corresponding files in your newly created Xcode project (Necessary files include: Expense, Recipe, Budget, Calculator, Recipes, ContentView, and Ice_CreamApp)
4. Launch the App
